# Руководство администратора VideoGrace Server

Центральным узлом системы является сервер. Он хранит учетные записи пользователей, настройки конференций, права доступа, передает и мультиплицирует трафик. Сервер может быть установлен на специализированную серверную машину или на обычный ПК.
