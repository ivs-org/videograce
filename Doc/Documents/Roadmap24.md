# VideoGrace Дорожная карта 2024
Стратегически, система должна стать глобальным распределенным средством связи, обладающей тысячами нод принадлежащих и управляемых как
ООО "ИВС" так и независимыми специалистами или компаниями.

## Сервер
Основное направление развития системы должно быть направлено на backend.
Сервер должен научиться быть нодой распределенной сети, так, чтобы обеспечить связность подключенных к разным серверам пользователей.
Также сервер должен предоставлять оптимальное websocket api для максимально легкого создания js клиентов для браузера.

### Новый Web UI админки сервера
 - REST для всего функционала сервера
 - Система управления токенами
 - Система webhook / sh hook для интеграции с внешними системами
 - Система ограничения прав для админа сервера (для облачного применения)
 - BLOB хранилище
 - Node integration API
 - Документация

## Клиент
Дальнейшее развитие клиентской части, должно идти преимущественно по направлению максимально удобного и понятного интерфейса пользователя.
Приложение должно вызывать отклик "на кончике пальцев"

## SDK
Приоритет в работе должен быть направлен на создание максимально логичного SDK с минимальным оверхедом. 
Документирование должно идти _перед_ кодированием. Создание минимальных приложений примеров должно предшествовать созданию 
функциональности в официальных клиентах.

### Работы
